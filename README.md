# Math_Quantum

Personal academic portfolio site for mathematics and quantum computing, built
with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme and
deployed with GitHub Pages.

**Live site:** https://amrutavelamuri.github.io/Math_Quantum/

## Featured project

**Quantum vs. Classical ML for Parkinson's Detection** — comparing a Variational
Quantum Classifier (PennyLane) to classical SVM/neural-network baselines for
detecting Parkinson's disease from gait data.
